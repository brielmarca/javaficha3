# Java Exercise Menu

Console application that groups introductory Java exercises behind an interactive menu. The project practices methods, numerical formulas, input handling and control flow.

## Exercises

- Calculate the perimeter of a square
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Calculate travel time from distance and average speed

The menu allows the user to complete an operation and return without restarting the program.

## Learning goals

- Organize logic into small methods
- Validate console input
- Use loops and selection structures
- Apply formulas with clear units
- Keep interface code separate from calculations

## Requirements

- JDK 17 or newer

## Compile and run

From the directory containing the source file:

    javac Main.java
    java Main

If the main class has a different filename, use that class name in both commands.

## Status
